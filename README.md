# matlab
MATLAB Visualization of multimodal bio signals in ICU dashboards

**Project Overview**
This project presents a MATLAB-based simulation and visualization system for multimodal biosignals commonly monitored in Intensive Care Units (ICU).
The system integrates multiple physiological signals (ECG, SpO₂, Respiratory signals) into a unified dashboard interface, demonstrating how biomedical signal processing and visualization techniques are applied in real-time patient monitoring systems.

**🎯 Objectives**
a. Simulate and process multimodal biosignals
b. Apply digital filtering to remove noise and artifacts
c. Extract vital physiological parameters
d. Design an ICU-style monitoring dashboard
e. Implement basic alert detection for abnormal conditions

**🛠️ Implemented Modules**
🔹** 1. Signal Acquisition & Preprocessing**
         Simulated ECG, SpO₂, and respiratory signals
         Removed baseline drift and noise using digital filters
         Normalized signals for consistent visualization

**🔹 2. Multimodal Visualization Dashboard**
         Real-time plotting of multiple biosignals
         Synchronized time axis across signals
         ICU-style layout for monitoring
         Dynamic updating of waveform displays

**🔹 3. Feature Extraction & Alerts**
           Heart Rate (HR) estimation from ECG
           Respiratory Rate (RR) calculation
           SpO₂ monitoring
           Threshold-based abnormality detection
           Simple alert system for critical values

**🧠 Key Concepts Applied**
      Biomedical Signal Processing
      Digital Filtering (Low-pass, High-pass)
      Feature Extraction
      Time-domain Analysis
      MATLAB Data Visualization
      Real-time Monitoring System Design
